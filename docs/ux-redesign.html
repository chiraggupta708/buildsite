✅ UX Improvements in this Design

  • Collapsible sidebar sections — Dashboard, Clients/Sites, Labour, Recent Sites. No more greyed-out Dashboard that never highlights.
  • Client list shows sites inline — Click a client to expand and see their sites right there. Avoids clicking into each client page.
  • Site detail = collapsible sections — Phases, Labour, Materials all visible inline. No more "Go to Phases" button that hides everything.
  • Phase table — See estimate amount, paid amount, payment status at a glance. "View" button to drill into details.
  • Labour cards with quick actions — Edit and Payment buttons visible without clicking into each labourer.
  • Materials with inline low-stock alerts — Red highlighted when stock is low. Log usage button right there.
  • Icons + microdata — Everything scannable without reading every word.
  • Sidebar shows "Recent Sites" — Jump directly to a site without going through Clients.
